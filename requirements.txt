# Python 3.10+ is required. The codebase uses PEP 604 union syntax
# (e.g. `str | None`) and PEP 585 generic builtins (`list[str]`,
# `dict[str, Any]`), neither of which work on 3.9 or earlier.
#
# Install with:
#   python3 -m pip install -r requirements.txt
#
# Note: this installs the Ollama *Python client library*, not the
# Ollama daemon itself. The daemon is a separate install from
# https://ollama.com/download. After installing it, start the local
# API and pull a model with:
#   ollama serve
#   ollama pull gemma4:31b
# =====================================================================

# Ollama Python client - the only third-party runtime import in the
# codebase (used in ollama.py for the transport layer). Pinned to a
# version with stable native tool-calling support, which the agent
# loop in agent_loop.py relies on.
