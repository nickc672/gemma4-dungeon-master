WHAT THIS IS

This folder is the development trail that led to the current codebase.
None of these notebooks are loaded or run by the live game.

Think of this folder as the project's sketchbook.
Each notebook prototypes one piece of the architecture in isolation, with verbose tracing turned on, before that piece was lifted into the proper Python package.
They are useful for understanding why the code is shaped the way it is, but the patterns they demonstrate have all been re-implemented inside "orchestrator/" - and the notebooks themselves are not on any active request path.
